# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in `mipiti-verify`, please report it responsibly.

**Email**: security@mipiti.io

Please include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact

We will acknowledge receipt within 48 hours and aim to provide a fix or mitigation within 7 days for critical issues.

## Scope

This policy covers the `mipiti-verify` package and its dependencies. For vulnerabilities in the Mipiti platform itself (api.mipiti.io), use the same email address.

## Supported Versions

Only the latest release receives security updates.
